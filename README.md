Customer Sales Analysis is a data analysis project that studies customer purchasing behavior and sales trends. The goal of this project is to analyze customer data and visualize important insights using a dashboard. This helps businesses understand customer spending patterns, popular products, and overall sales performance.

Objective

The main objective of this project is to:

Analyze customer purchase data.

Identify sales trends and patterns.

Understand customer demographics and spending behavior.

Create a dashboard for better visualization of insights.

Dataset

The dataset contains customer-related sales information such as:

Customer ID

Age

Gender

Income

Product Category

Purchase Amount

Payment Method

Season

Shipping Type

Subscription Status

Tools and Technologies

The following tools were used in this project:

MySQL – For storing and managing the dataset.

Power BI – For data visualization and dashboard creation.

CSV File – For importing the dataset.

SQL – For querying and analyzing data.
